# fnp-Sales-data-analysis-using-MySql-and-Python-
Using MySql and Python libraries
This project provides a comprehensive end-to-end analysis of sales data for a retail environment (fnp). By integrating MySQL for robust data storage and Python for advanced analytics and visualization, the project transforms raw transactional data into actionable business intelligence
- Analyze customer behavior
- Product performance
- sales trends to identify growth opportunities.
- Designing a relational database
- Performing ETL operations.

Database : MySQL (Relational Schema Design, Complex Joins, CTEs, Window Functions)
Language : Python 
Libraries : Pandas: For data manipulation and cleaning.
            SQLAlchemy & PyMySQL: For establishing a seamless connection between Python and MySQL.
            Matplotlib & Seaborn: For generating insightful data visualizations.
